# Web App

Nextjs web appplication framework that leverages keystonejs as a data engine

## Getting Started

1. install modules
2. run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:7777](http://localhost:7777) with your browser to see the frontend web app.

Open [http://localhost:3333](http://localhost:3333) with your browser to see the backend keystone admin ui.
